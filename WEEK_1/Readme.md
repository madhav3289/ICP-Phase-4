# Week 1 — Arrays & Two Pointers

## Topics Covered
- Arrays: traversal, hashing, prefix ideas
- Two Pointers: opposite ends, moving windows, deduping

## Problems
- buy_and_sell_stock.java: Max profit from single buy/sell (one pass).
- count_primes.java: Count primes below n (sieve of Eratosthenes).
- first_missing_positive.java: Smallest missing positive using in-place marking.
- first_occurence_index.java: Find first occurrence of a pattern in a string.
- median_of_two_sorted_array.java: Median via partition/binary search on arrays.
- three_sum.java: Unique triplets summing to zero (sort + two pointers).

## How To Run (Windows / Java)
```bash
# From repository root
cd WEEK_1
javac three_sum.java
java three_sum
```

Replace `three_sum` with any file’s classname to run other solutions.
